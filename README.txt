# Sports Science & Performance Data Portfolio

This repository contains the source code for my interactive web application, designed to showcase my skills at the intersection of Applied Sports Science, Data Engineering, and Performance Analytics.

**Live Application:** [portfolio-pierre-biache.streamlit.app](https://portfolio-pierre-biache.streamlit.app/)

## Objective

My primary motivation as a Sports Scientist is to engage in meaningful work where data-driven insights have a direct, tangible impact on the court. This portfolio demonstrates my ability to manage the entire data pipeline—from raw sensor collection to actionable decision-support tools for coaching and medical staff.

## Project Highlights

### 1. Professional Experience Implementations
* **AC Sparta Prague:** Development of automated Match and Training reports using Power BI, integrating STATSports GPS and VALD Diagnostics data.
* **Stats Perform:** Research on Large Language Models applications for autonomous sports data analysis and construction of ETL pipelines using AWS and Terraform.

### 2. Basketball Trajectory & Physical Load Analysis
* **Context:** Processing high-frequency optical tracking data to monitor player movement and intensity.
* **Technology:** Python (Pandas, NumPy), Gzip compression for large-scale data handling.
* **Insights:** Calculation of instantaneous speeds, anaerobic zone distribution, trajectories, and phase-based movement analysis.

### 3. Ligue 1 Scouting Dashboard
* **Context:** Interactive tool for player profiling and talent identification in professional soccer.
* **Technology:** Scikit-learn, Plotly, Streamlit.
* **Insights:** Comparative performance metrics and radar charts to support scouting departments.

## Technical Stack

* **Languages:** Python (Advanced), SQL, R, Matlab.
* **Data Engineering:** ETL Pipelines, AWS Cloud, Terraform, Data Governance.
* **Sports Technologies:** STATSports, Catapult, VALD (ForceDecks, NordBord), Optical Tracking.
* **Visualization:** Power BI, Plotly, Streamlit.

## Research & Academic Background

* **Master’s Degree:** Sport Science and Human Movement Engineering (Aix-Marseille University).
* **Core Focus:** Biomechanics, Exercise Physiology, and Data Science.
* **Notable Work:** Evaluation of AI-generated code accuracy for expert-level sports performance analysis / Thesis about the influence of biological age on physical performances in youth football.
